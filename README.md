# Welcome to Star wars!
Star wars fun app using https://swapi.co/

# Install
	clone the repo
	npm install
	
## Production Build

npm run build
npm run serve

## Dev server

npm start

## Usage
	=> Login Details
		Luke Skywalker/19BBY  // User with no search limit
		Leia Organa/19BBY  // User with search limit of 15 results in a minute

	=> Search
		Search planets
			Alderaan
			Yavin
